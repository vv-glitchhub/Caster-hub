# Caster OS Roadmap

Caster OS is a premium personal AI operating system prototype that connects life, wealth, health, gaming and decision intelligence into one ecosystem.

## Current foundation

- Premium public pages
- Global navigation
- Admin console
- Vercel Analytics
- Local memory layer
- Memory service abstraction
- Supabase readiness status
- Supabase database schema
- Agent context display

## Phase 1: Launch polish

- Reduce visual noise across public pages
- Keep each page focused on 3 to 5 visible concepts
- Improve mobile spacing
- Remove prototype-only copy from public pages
- Capture launch screenshots

## Phase 2: Supabase memory

- Run `database/schema.sql` in Supabase
- Add `NEXT_PUBLIC_SUPABASE_URL`
- Add `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- Connect memory service to real tables
- Add profile-based memory loading
- Add memory creation and updates

## Phase 3: Agent engine

- Build prompt context from memory service
- Add recommendation generation
- Add decision scoring
- Add confidence and risk explanation
- Add action history

## Phase 4: Product system

- Account system
- Private dashboard mode
- Admin analytics view
- Widget marketplace concept
- Caster Core API

## Design principle

Premium means less noise, stronger hierarchy and more space.

Every page should answer:

1. What is this module?
2. Why does it matter?
3. What should the user do next?
